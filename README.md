# StudyBloom

StudyBloom is a desktop study analytics application developed using Python. The application helps students record study sessions, monitor productivity, and analyze study habits through data visualization and performance metrics.

## Features

- Record and store study sessions
- Track hours studied for different subjects
- Record focus scores for each session
- Save study dates
- View all study records
- Analyze study statistics
- Generate subject-wise study graphs
- Track study streaks
- Delete study sessions

## Technologies Used

- Python
- Tkinter (Graphical User Interface)
- SQLite (Database Management)
- Matplotlib (Data Visualization)
